# Structur
Ett projektledningssystem baserat på Kanban och annan Lean-metodik. Endast frontend är implementerad. Jag skapade det med hjälp av plattformen Lovable.

För att köra projektet efter att ha synkat detta repo, kör följande kommand i terminalen:

```bash
npm install
npm run build
npm run preview
```
